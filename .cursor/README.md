# Configuração do Cursor para QA Agile Guide

Este diretório contém configurações específicas para o Cursor AI se adaptar melhor ao projeto.

## Arquivos de Configuração

- `.cursorrules` - Regras e padrões do projeto
- Este README - Documentação das configurações

## Como o Cursor Usa Isso

O Cursor AI lê automaticamente o arquivo `.cursorrules` na raiz do projeto para:
- Entender o contexto do projeto
- Seguir padrões de código estabelecidos
- Sugerir código alinhado com as convenções
- Ajudar com decisões arquiteturais

## Atualizações

Quando o projeto evolui, atualize o `.cursorrules` para refletir:
- Novas dependências
- Mudanças arquiteturais
- Novos padrões estabelecidos
- Convenções atualizadas
